# Latest Update in [[Zeddal]] App: Status Bar Addition

## Overview
The latest enhancement in the [[Zeddal]] application is the introduction of a **status bar** located at the bottom of the interface. This feature is designed to improve user interaction and accessibility by providing real-time status updates.

## Features of the Status Bar
- **Shape and Design**: The status bar features a pill-shaped design, which is both modern and minimalist, ensuring it blends seamlessly with the overall interface of the app.
- **Functionality**:
  - **Indicator**: It includes an indicator that displays the current status ('ready' state), informing users that the app is prepared for interaction.
  - **Speaking and Recording Logs**: The bar shows dynamic updates related to speaking duration and recorded text logs, allowing users to monitor their activity within the app efficiently.

This addition aims to enhance user experience by providing essential information in a concise and accessible format at a glance.
