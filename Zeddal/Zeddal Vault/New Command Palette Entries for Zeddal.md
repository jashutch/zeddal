# New Command Palette Entries for [[Zeddal]]

We have introduced new command palette entries for [[Zeddal]], enhancing its functionality and user experience. Below are the details of the newly added commands:

## Added Commands

1. **Start Voice Recording**
   - Activates the voice recording feature, allowing users to capture their spoken input directly into the system.

2. **Browse Recording History**
   - Enables users to view and access their past recordings, facilitating easy retrieval and review of previous inputs.

3. **Inspect Contextual Links in Current Note**
   - Allows users to examine all contextual links within the active note, aiding in efficient navigation and reference checks within their digital knowledge base.
