# New Features in [[[Zeddal]]](https://zeddal.ai): Magic Moment Update

In the latest update to [[Zeddal]], several exciting new features have been introduced to enhance user experience:

- **Link Integration**: Users can now create links directly to existing Zeddal nodes within their documents. This integration facilitates smoother navigation and improved organization within the Zeddal ecosystem.

- **Wiki Link Counter**: Zeddal now includes a feature that counts the number of wiki links added throughout a document. This tool helps users track their progress and measure the extent of AI's involvement in document creation.
