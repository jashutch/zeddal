# Upgraded Media Control Unit in [[Tesla Model S]] Plaid (2022 - Recent Refresh)

## Overview
The [[Tesla Model S]] Plaid has been equipped with an advanced media control unit from 2022 up to its latest refresh. This note summarizes the key features and enhancements included in this upgrade.

## Display and System Specifications
- **Screen Size and Resolution**: The main touchscreen is 17 inches, landscape-oriented, with a resolution of approximately 2200 by 1300 pixels.
- **Infotainment System**: Powered by a new system utilizing an AMD or Navi 23-style GPU.
- **Computing Power**: The system delivers about 10 teraflops, enabling capabilities akin to console-level gaming, high-resolution media streaming, and a more responsive user interface (UI).

## Connectivity and Hardware Features
- **USB-C Ports and Wireless Charging**: The unit includes multiple USB-C ports and wireless charging pads for both front and rear passengers.
- **Internet Connectivity**: Enhanced with Wi-Fi and LTE support, catering to high bandwidth functions like video streaming and game controller inputs.

## User Interface Enhancements
- **Revamped UI**: The UI has undergone significant revisions, featuring a more flexible layout that allows for resizing and repositioning of the media panel alongside the map.
- **App Integration**: Functions are now organized into apps, media, navigation, and games, improving accessibility and user interaction.

## Passenger Experience
- **Rear Seat Touchscreen**: A dedicated touchscreen for rear passengers is provided, enhancing control over media playback.

This upgrade significantly enhances the in-car entertainment and connectivity options, aligning with Tesla's commitment to combining luxury with cutting-edge technology as seen in the overall design and features of the Tesla Model S Plaid.
